# 🎯 Number Guessing Game (Java Swing)

A GUI-based number guessing game built using Java Swing.  
The player selects difficulty and tries to guess a randomly generated number within limited attempts.

---

## 🚀 Features

- 🎮 3 Difficulty levels (Easy / Moderate / Hard)
- 🧠 Smart scoring system
- 🏆 High score tracking
- 📊 Win / Loss counter
- 🔄 New game reset functionality
- 💻 Swing-based GUI
